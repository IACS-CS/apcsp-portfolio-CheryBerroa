What is the Binary Project About

In this assignment, you make a game where the computer guesses what you're thinking by asking yes/no questions and narrowing down the options. It keeps halving the possibilities based on your answers. You’ll use if statements and loops to control the game and handle inputs/outputs to talk to the player. You also test the game to make sure it works properly and only accepts valid answers.

Peer to Peer filesharing

Shawn Fanning changed how people share files by creating Napster, a program that allowed users to share music directly with each other. His idea of sharing files without a central server changed the way we think about downloading and sharing content online. Although Napster faced legal issues, Fanning’s work led to the development of other peer-to-peer platforms that are still used today.

Turtle Mini Drawing

In this project,You were supposed to create a program that allows users to customize a drawing through commands. The drawing should be divided into parts, with each part handled by a separate function. Each function should take parameters (like size or shape) to customize the drawing. Every group member was responsible for writing at least one function with meaningful parameters. The program should allow the user to input commands, use at least three functions to create the drawing, and combine them to generate a final customized image.

Experiance

For this assignment, I created a game where the computer guesses the number you're thinking of using binary search. The computer asks if the number is higher or lower than a guess, then narrows down the options until it finds the right one. I used if statements to check the user's answers and loops to keep guessing until the right number was found. I also tested the game to make sure it only accepted valid inputs, like numbers, and gave clear feedback to the user. This helped me understand how binary search works and how to build a program that interacts with the user.
