Experiance

For this assignment, I created a game where the computer guesses the number you're thinking of using binary search. The computer asks if the number is higher or lower than a guess, then narrows down the options until it finds the right one. I used if statements to check the user's answers and loops to keep guessing until the right number was found. I also tested the game to make sure it only accepted valid inputs, like numbers, and gave clear feedback to the user. This helped me understand how binary search works and how to build a program that interacts with the user.
