Overview

In this project, my group and I created a game from scratch based on chocolate bars. The game determines which chocolate bar you are most like based on your answers to a series of questions.

How It Works

The game begins by asking the user for their name. Then, it presents a series of questions related to personal preferences, such as favorite colors, seasons, and activities. Each question has multiple-choice answers that guide the user toward discovering their ideal chocolate bar.

HTML Structure

The HTML is what builds the structure of the game. It sets up the main part of the page where players will see the quiz questions and results. The HTML connects to the main.js file, which makes the game interactive.

CSS Style

The CSS is what makes the game look nice. It gives the page a dark background and a clean layout, so players can easily focus on answering the questions. The page is also designed to look good on any screen size, whether on a phone or computer.

main.js: JavaScript Logic

The main.js file is the brain of the game. It starts by asking for your name and then takes you through a series of fun questions. Based on your answers, it uses binary search to figure out which chocolate bar you match with.

Once the quiz is done, you get a fun result showing which chocolate bar matches your personality. You can choose to play again or leave, with a thank-you message at the end.
