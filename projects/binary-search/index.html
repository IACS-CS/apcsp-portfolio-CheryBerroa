Overview

In this project, my group and I created a game from scratch based on chocolate bars. The game determines which chocolate bar you are most like based on your answers to a series of questions.

How It Works

The game begins by asking the user for their name. Then, it presents a series of questions related to personal preferences, such as favorite colors, seasons, and activities. Each question has multiple-choice answers that guide the user toward discovering their ideal chocolate bar.

HTML

The HTML for our chocolate bar quiz game, "Vite App," sets up the overall structure. It features a main content area called "app," where players will see the quiz questions and results. This document also connects to main.js, the file that brings the game to life with interactivity.

Style

Our CSS styles the game to create an inviting experience. By removing margins and padding, we achieve a clean layout. The dark gray background makes the text pop, while a grid layout centers everything nicely on the page. This way, players can focus on enjoying the quiz without distractions.
