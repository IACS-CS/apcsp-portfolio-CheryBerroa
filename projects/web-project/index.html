Project: Peer to Peer Filesharing

Back to Portfolio

In this project, we researched a technology that impacts the web, specifically focusing on Peer-to-Peer (P2P) file sharing. We explored how P2P networks allow direct file sharing between users, without the need for central servers. This decentralized approach improves speed and efficiency, allowing users to share large files more easily and securely.

HTML was used to embed the presentation and structure the page where we displayed our findings. The embedded Google Slides presentation helped communicate the details of P2P file sharing and its significance on modern web technologies. CSS was minimally used, as the focus was more on the content and presentation of the research. However, the overall layout and design were kept clean to ensure readability.

While JavaScript was not central to this project, understanding its role in web technologies helped us see how file-sharing systems can be implemented in web applications. For instance, JavaScript could be used to facilitate file transfers, handle connections, or improve the user interface of a P2P file-sharing platform.

Through this project, we learned how Peer-to-Peer technology works and its impact on the efficiency and scalability of web applications. We also gained experience in researching and presenting technology topics, and how different technologies (like HTML, CSS, and JavaScript) can support a modern web system.
